What is the NT.gov.au Design System?

The NT.gov.au Design System provides a single source of truth for styles, components, and patterns when creating digital experiences. It supports both React + TypeScript and vanilla JavaScript for CMS integrations.

It includes a public documentation site, live Storybook examples, and design tokens for consistent theming.

Out-of-the-box accessibility

Components target WCAG 2.1 AA compliance with accessible color variants and focus states.

Who is it for?

Designed for NT Government teams to reduce time and cost while improving quality:

  • Designers — rapidly compose prototypes with proven patterns.
  • Developers — build pixel-perfect UIs with comprehensive docs and TS types.
  • Content creators — assemble pages that match production components.
  • Product owners — review consistent experiences and iterate faster.

Key features

  • 35+ reusable components across navigation, forms, display, and feedback.
  • 200+ design tokens for colors, spacing, typography, and radii.
  • Multiple themes: NT.GOV.AU and NTG Central.
  • Responsive, mobile-first layout patterns.
  • Live Storybook with interactive examples.
